# JavaScript and React Learning Notes Index

This note set is designed for someone starting from zero in JavaScript and React.

The learning path is intentionally split into two levels:

- Basic level: surface understanding, syntax, common usage, simple implementation
- Intermediate level: deeper behavior, patterns, tradeoffs, common mistakes, interview readiness

## Best reading order

1. `README-01-Overview-And-Mind-Map.md`
2. `README-02-JavaScript-Basics.md`
3. `README-03-JavaScript-Intermediate-And-Deeper-Topics.md`
4. `README-04-React-Basics.md`
5. `README-05-React-Hooks-Deep-Dive.md`
6. `README-06-React-Intermediate-Patterns.md`
7. `README-07-Practice-Questions-And-Mini-Exercises.md`

## How to use these notes

- First pass: read everything once in order.
- Second pass: type the examples by hand and run them.
- Third pass: build tiny examples for each hook and concept.
- Fourth pass: revisit the question file and answer without looking.

## Main goal

By the end of these notes, you should be able to:

- understand the important parts of JavaScript needed for React
- read and write basic to intermediate React code
- understand hooks and when to use them
- explain concepts in interviews
- build small apps and understand larger existing codebases

## Important principle

Do not try to memorize all syntax first.

Learn in this order:

1. Understand the concept.
2. See a simple example.
3. See where it is used in real code.
4. Write your own version.
5. Explain it in your own words.

That is the fastest path to actually learning JavaScript and React.
